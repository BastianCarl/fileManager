 412 Precondition Failed
    Google will not index a URL with the HTTP 412
    status code indicates that access to the target resource was denied
    conditional requests
    Condition is defined using If-Unmodified-Since or If-Match headers
    used for updating a cache

ETag
    an identifier for a specific version of a resource
    lets caches be more efficient and save bandwidth
    used for If-Match and If-None-Match

428 Precondition Required
    Typically, a 428 response means that a required precondition header such as If-Match is missing.
